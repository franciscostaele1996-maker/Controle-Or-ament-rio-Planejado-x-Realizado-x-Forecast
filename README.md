# Controle-Or-ament-rio-Planejado-x-Realizado-x-Forecast
Controle Orçamentário Planejado x Realizado x Forecast


              Elaborei um modelo de controle orçamentário com análise de variações mensais, comparando:

Planejado
Realizado
Forecast
Variação em valor e %

O objetivo foi simular uma estrutura real de FP&A, com foco em análise gerencial e tomada de decisão.

Principais insights da análise:

• Em Janeiro, houve economia de 31% vs orçamento

• Em Fevereiro, a variação caiu para 23%

• Em Março, o desvio reduziu para 14%

• Em Abril, o forecast já indica apenas 8% de diferença

                        Isso demonstra um cenário de:

 Redução gradual da variação

 Maior previsibilidade financeira

 Melhor alinhamento entre planejamento e execução

              Além disso, segmentei por centro de custo:

Produção/Industrial

Comercial/Vendas

Permitindo análise mais estratégica por área e olhando num contexto geral .

                      

Mais importante do que apenas acompanhar orçamento é:

• Entender o comportamento das despesas
• Antecipar desvios com forecast
• Gerar informação para decisão

Controle orçamentário não é sobre cortar custos.
É sobre inteligência financeira.
